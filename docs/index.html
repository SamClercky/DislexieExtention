Dyxitool

Textadaptor for dyslexics

Welcome

Welcome to the site of a beautiful project. This project is opensource and contains (for now) only an extention for chrome The extention helps people who have dyslexia; it uses the OpenDyslexic-font, an award winning invention made for this purpose. With this extention you can change all the text on a webpage in the OpenDyslexic-font. Also a colored background can help better reading so you can choose a color and use it as the background color of the text when you hover it with the cursor.

Customize everything

We spend a lot of time to make everything easy to use and gave you the ability to change averything like you prefer.

Features

The extension provides for now on only two main features:

OpenDyslexic-font

Change all the font to OpenDyslexic-font

Special background

When you hover text the background will change so you can read it more easily. You can customize the color to wathever you like
